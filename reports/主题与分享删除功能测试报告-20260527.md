# 主题与分享删除功能 — 测试报告

> 日期: 2026-05-27 | 结果: **通过**

---

## 功能概述

- **主题删除**：从数据库删除主题记录，不影响实际文件
- **分享链接撤销**：`active=False`，保留记录可查
- **分享链接删除**：从数据库彻底移除记录

---

## API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| `DELETE` | `/api/themes/{theme_id}` | 删除主题（仅删 DB 记录） |
| `DELETE` | `/api/shares/{share_id}` | 撤销分享（active=False） |
| `DELETE` | `/api/shares/{share_id}/delete` | 删除分享（彻底移除） |

## 前端变更

### `console.js`
- 主题操作列增加"删除"按钮（红色）
- 分享操作列增加"删除"按钮（红色，仅 active 状态显示）
- `deleteTheme(id, title)` — 确认后调用 DELETE API
- `deleteShare(id)` — 确认后调用 DELETE /delete API

---

## 注意事项

- 删除主题仅删除数据库记录，不会删除磁盘上的文件
- 删除分享后该链接立即失效，无法恢复
- 两个操作都有确认弹窗防止误操作
