# 路由权限重构 + 主题Logo上传 + 第三层权限修复 报告

**日期**: 2026-05-27

## 一、路由级权限重构

### 问题
匿名用户拿到 `/s/{token}` 后获得持久 share_token cookie，可直接访问 `/content/{path}`，保存 URL 后即使撤销分享也无法阻止访问。

### 解决方案
| 变更 | 之前 | 之后 |
|------|------|------|
| 主页点击主题 | `/content/themes/xxx.html` 暴露真实路径 | `/t/{slug}` 不暴露路径 |
| 分享入口 `/s/{token}` | 设持久 cookie | 注入 `<base href="/s/{token}/file/">` 标签，无 cookie |
| 分享内导航 | 相对链接 → `/content/xxx`（cookie 判断） | 相对链接 → `/s/{token}/file/xxx`（实时查库） |
| `/content/{path}` 匿名 | cookie 判断，持久钥匙 | 实时判断：ContentRule → Theme.accessible |

### 新增路由
- `GET /t/{slug}` — 主题入口（不暴露文件路径）
- `GET /s/{token}/file/{path:path}` — 分享内容访问（每次实时校验）
- `GET /logos/{filename}` — Logo 文件服务

### 测试结果
- A: /s/{token}/file/ 主题页面 → 200 ✅
- B: /s/{token}/file/ 子页面（无规则，allow_content=true）→ 200 ✅
- C: ContentRule=hidden 阻止分享 → 302 ✅
- D: ShareFileRule=hidden 阻止 → 302 ✅
- E: ShareFileRule=public 允许 → 200 ✅
- F: ContentRule=hidden + ShareFileRule=public → 302（全局优先）✅
- G: 撤销分享 → 404 ✅
- H: /content/ 匿名 + theme accessible → 200 ✅
- I: /content/ 匿名 + theme accessible=False → 302 ✅
- J: /t/{slug} accessible=False → 302 ✅
- K: Admin /content/ 正常工作 → 200 ✅
- L: /s/{token} 注入 `<base>` 标签 → ✅
- M: /t/{slug} 注入 `<base href="/content/">` → ✅

## 二、主题 Logo 上传 + Emoji 选择

### 功能
1. 控制台编辑主题时可上传图片（PNG/JPG/SVG/WebP/GIF，最大 2MB）作为 Logo
2. 可从预设 30 个 Emoji 中选择
3. 首页主题卡片显示上传的 Logo 或选择的 Emoji（默认 🚀）

### 技术实现
- 复用 `Theme.icon` 字段：`logo:文件名` 表示图片，其他值表示 emoji
- 新增端点：`POST/DELETE /api/themes/{theme_id}/logo`
- Logo 存储在 `data/logos/` 目录
- 前端新增 Logo 预览、上传按钮、删除按钮、Emoji 网格选择器

### 测试结果
- 上传 Logo → API 返回 logo_url → 首页显示图片 ✅
- 选择 Emoji → API 返回 icon → 首页显示 emoji ✅
- 删除 Logo → logo_url 为 null → 首页显示默认 🚀 ✅

## 三、第三层文件权限修复

### 问题
`share_file_rules` API 使用 `resolve_file_theme` 只能匹配到 `themes/` 下的文件，`presentations/`、`references/`、`thinking/` 等第三层文件不会出现在分享文件权限列表中。

### 解决方案
修改 `list_share_files` 逻辑：不再过滤为"只属于此主题的文件"，改为列出内容目录下**所有**文件（仅排除属于其他主题的文件）。无明确主题归属的文件默认遵循分享的 `theme.accessible` 设置。

### 测试结果
- 分享文件权限列表现在包含 22 个文件（含 presentations/ 等）✅
- 对某个分享设置 presentation=hidden，仅该分享受影响 ✅
- 其他分享不受影响 ✅
